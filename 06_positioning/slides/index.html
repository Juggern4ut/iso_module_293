Positionierung in CSS

Mit CSS kann festgelegt werden, wo sich ein Element auf der Webseite befindet.

  • Standardmässig folgen Elemente dem normalen Dokumentenfluss
  • Mit position kann dieses Verhalten geändert werden

Die Property position

Die CSS-Property position bestimmt die Art der Positionierung eines Elements.

Wichtige Werte:

  • static
  • relative
  • absolute
  • fixed

Statische Positionierung

position: static ist der Standardwert.

  • Elemente folgen dem normalen Dokumentenfluss
  • top, left, right, bottom haben keine Wirkung
  • Keine manuelle Verschiebung möglich

Beispiel: static

                    
                        .box {
                        position: static;
                        }

Das Element erscheint genau dort, wo es laut HTML-Struktur vorgesehen ist.

Relative Positionierung

position: relative positioniert ein Element relativ zu seiner ursprünglichen Position.

  • Element bleibt im Dokumentenfluss
  • Kann visuell verschoben werden
  • Der ursprüngliche Platz bleibt reserviert

Beispiel: relative

                    
                        .box {
                        position: relative;
                        top: 20px;
                        left: 30px;
                        }

Das Element wird verschoben, aber andere Elemente verhalten sich, als wäre es noch an der ursprünglichen Stelle.

Absolute Positionierung

position: absolute entfernt das Element aus dem normalen Dokumentenfluss.

  • Positioniert relativ zum nächsten positionierten Elternelement
  • Falls keines existiert: relativ zum Body
  • Belegt keinen Platz mehr im Layout

Beispiel: absolute

                    
                        .container {
                        position: relative;
                        }

                        .box {
                        position: absolute;
                        top: 10px;
                        right: 10px;
                        }

Das Element wird innerhalb des Containers exakt positioniert.

Fixe Positionierung

position: fixed positioniert ein Element relativ zum Browserfenster.

  • Bleibt beim Scrollen sichtbar
  • Wird aus dem Dokumentenfluss entfernt
  • Typisch für Header oder Buttons

Beispiel: fixed

                    
                        .header {
                        position: fixed;
                        top: 0;
                        left: 0;
                        width: 100%;
                        }

Der Header bleibt immer am oberen Rand des Bildschirms sichtbar.

Vergleich der Positionierungen

Position Im Dokumentenfluss
static Ja
relative Ja
absolute Nein
fixed Nein

Was ist z-index?

Mit z-index wird festgelegt, welches Element im Vordergrund liegt.

  • Gilt nur für positionierte Elemente
  • Höherer Wert = weiter vorne
  • Nützlich bei überlappenden Elementen

Beispiel: z-index

                    
                        .box1 {
                        position: absolute;
                        z-index: 1;
                        }

                        .box2 {
                        position: absolute;
                        z-index: 2;
                        }

box2 liegt vor box1, da der z-index höher ist.

  • relative verschiebt Elemente ohne Layoutbruch
  • absolute positioniert frei innerhalb eines Containers
  • fixed bleibt immer sichtbar
  • mit dem z-index kann die Überlagerungsreihenfolge von positionierten Elementen festgelegt werden
