03 - CSS Layout Grundlagen

Box-Modell & Inline / Block

Lernziele dieser Lektion

  • Verstehen, wie das CSS Box-Modell aufgebaut ist.
  • Unterschiede zwischen Inline- und Block-Elementen kennen.
  • Wissen, wie sich Box-Modell und Display-Typ auf das Layout auswirken.

Rückblick

  • CSS bestimmt das Aussehen von HTML-Elementen.
  • Elemente werden über Selektoren angesprochen.
  • Jedes Element besitzt ein bestimmtes Layout-Verhalten.

Was ist das Box-Modell?

  • Jedes HTML-Element wird als rechteckige Box dargestellt.
  • Diese Box besteht aus mehreren Bereichen.
  • Das Box-Modell ist zentral für Layout und Abstände.

Aufbau des Box-Modells

  • Content – eigentlicher Inhalt (Text, Bild)
  • Padding – Innenabstand
  • Border – Rahmen
  • Margin – Aussenabstand

Box-Modell im Code

            
            .box {
              width: 300px;
              padding: 20px;
              border: 2px solid black;
              margin: 30px;
            }

Die display-Eigenschaft

  • Bestimmt, wie ein Element im Layout dargestellt wird.
  • Die wichtigsten Werte:
    • block
    • inline
    • inline-block

Block-Elemente

  • Nehmen die gesamte Breite ein.
  • Beginnen immer in einer neuen Zeile.
  • Breite, Höhe, Margin und Padding funktionieren vollständig.

Beispiele:

div, p, section, header

Inline-Elemente

  • Belegen nur so viel Platz wie nötig.
  • Beginnen nicht in einer neuen Zeile.
  • Breite und Höhe können nicht gesetzt werden.

Beispiele:

span, a, strong, em

Inline vs Block

Eigenschaft Block Inline
Neue Zeile Ja Nein
Breite / Höhe Ja Nein
Margin / Padding Ja Eingeschränkt

inline-block

  • Kombiniert Eigenschaften von Inline und Block.
  • Elemente bleiben in einer Zeile.
  • Breite, Höhe, Margin und Padding funktionieren.
            
            .button {
              display: inline-block;
              padding: 10px 20px;
            }

Display-Typ ändern

Der Display-Typ eines Elements kann jederzeit mit CSS geändert werden.

            
            a {
              display: block;
            }

Typisches Layout-Problem

Warum funktioniert margin-top bei Inline-Elementen nicht wie erwartet?

Weil Inline-Elemente nicht das volle Box-Modell verwenden.

tldr

  • Jedes Element ist eine Box.
  • Das Box-Modell besteht aus Content, Padding, Border und Margin.
  • Block- und Inline-Elemente verhalten sich grundlegend unterschiedlich.
  • display steuert das Layout-Verhalten.
  • inline-block ist oft ein guter Kompromiss.
