CSS Flexbox

Moderne Layouts einfach und flexibel erstellen

Lernziele

  • Verstehen, was Flexbox ist und wann es verwendet wird
  • Container- und Item-Properties unterscheiden können
  • Elemente entlang Haupt- und Querachse ausrichten
  • Responsive Layouts mit Flexbox umsetzen

Was ist Flexbox?

Flexbox ist ein Layout-Modell in CSS, um Elemente in einer Zeile oder Spalte flexibel anzuordnen.

  • Ideal für Navigationen, Kartenlayouts und UI-Elemente
  • Einfaches Zentrieren von Inhalten
  • Sehr gut für responsive Layouts geeignet

Flex Container aktivieren

Ein Element wird zum Flex-Container mit display: flex. 

                    
                        .container {
                            display: flex;
                        }

Alle direkten Kind-Elemente werden damit zu Flex-Items.

Hauptachse und Querachse

  • Hauptachse (main axis): Richtung der Flex-Items
  • Querachse (cross axis): senkrecht zur Hauptachse
  • Die Achsenrichtung wird mit flex-direction festgelegt

flex-direction

Definiert die Richtung der Hauptachse.

                    
                        .container {
                            display: flex;
                            flex-direction: row;
                        }
  • row (Standard)
  • column
  • row-reverse
  • column-reverse

justify-content

Richtet Flex-Items entlang der Hauptachse aus. 

                    
                        .container {
                            display: flex;
                            justify-content: space-between;
                        }
  • flex-start, center, flex-end
  • space-between, space-around, space-evenly

align-items

Richtet Flex-Items entlang der Querachse aus. 

                    
                        .container {
                            display: flex;
                            align-items: center;
                        }
  • stretch (Standard)
  • flex-start, center, flex-end
  • baseline

Abstände mit gap

Mit gap werden gleichmässige Abstände zwischen Flex-Items erzeugt. 

                    
                        .container {
                            display: flex;
                            gap: 16px;
                        }

flex-wrap

Bestimmt, ob Items in die nächste Zeile umbrechen dürfen. 

                    
                        .container {
                            display: flex;
                            flex-wrap: wrap;
                        }
  • nowrap (Standard)
  • wrap
  • wrap-reverse

Eigenschaften von Flex-Items

Wichtige Properties für einzelne Items:

  • flex-grow: darf wachsen
  • flex-shrink: darf schrumpfen
  • flex-basis: Startgrösse
  • align-self: individuelle Ausrichtung

Shorthand: flex

Die Property flex kombiniert grow, shrink und basis. 

                    
                        .item {
                            flex: 1;
                        }

flex: 1 verteilt den verfügbaren Platz gleichmässig auf alle Items.

Beispiel: einfache Navbar

                    
                        nav {
                            display: flex;
                            justify-content: space-between;
                            align-items: center;
                        }

Logo links, Navigation rechts und alles vertikal sauber ausgerichtet.

Häufige Flexbox-Properties

Property Gilt für Zweck
justify-content Container Ausrichtung auf Hauptachse
align-items Container Ausrichtung auf Querachse
gap Container Abstand zwischen Items
flex Item Wachstum/Schrumpfen/Startgrösse

tldr

  • Flexbox macht Layouts flexibel und übersichtlich
  • Container steuert Ausrichtung, Items steuern ihr eigenes Verhalten
  • justify-content = Hauptachse, align-items = Querachse
  • Mit wrap und gap lassen sich responsive Layouts einfach bauen
