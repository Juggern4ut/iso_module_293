Saubere Struktur ist die Grundlage für jedes spätere Styling.

Repetition: Semantische Elemente

  • header, main, footer
  • nav für Navigationen
  • section für inhaltliche Bereiche
  • Semantik verbessert Lesbarkeit, SEO und Barrierefreiheit

Repetition: CSS Basics

                    
                        h1 {
                          color: #2e86de;
                          margin-bottom: 16px;
                        }

                        .card {
                          background: #f5f5f5;
                          padding: 12px;
                        }
  • Selektoren gezielt einsetzen
  • Mit Klassen wiederverwendbar stylen

Repetition: Box Model

  • content: eigentlicher Inhalt
  • padding: Innenabstand
  • border: Rahmen
  • margin: Aussenabstand
                    
                        .box {
                          padding: 16px;
                          border: 2px solid #fff;
                          margin: 20px;
                        }

Repetition: Best Practices

  • Einrückung und Struktur konsequent halten
  • Sinnvolle Klassennamen statt kryptischer Namen
  • Keine unnötigen verschachtelten div-Container
  • Code kommentieren, wenn Logik nicht sofort klar ist

Repetition: Positionierung

Wert Verhalten
static Standard, im normalen Dokumentenfluss
relative Verschiebung relativ zur eigenen Position
absolute Aus dem Fluss, relativ zum positionierten Elternteil
fixed Fix relativ zum Browserfenster

Repetition: Flexbox

                    
                        .container {
                          display: flex;
                          justify-content: space-between;
                          align-items: center;
                          gap: 12px;
                        }
  • justify-content steuert die Hauptachse
  • align-items steuert die Querachse

Praxisaufgabe

Erstelle eine kleine Webseite mit Header, Navigation, Hauptinhalt und Footer.

  • Semantische HTML-Struktur verwenden
  • Box Model sichtbar einsetzen (padding, margin, border)
  • Navigation mit Liste und Links erstellen
  • Layout mit Flexbox aufbauen
