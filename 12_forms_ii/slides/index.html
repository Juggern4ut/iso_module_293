Modul 293

HTML Formulare II

Mehr Auswahlmöglichkeiten und längere Eingaben

Lernziele

  • Dropdowns mit select und option erstellen
  • Radio-Buttons und Checkboxen unterscheiden
  • Mehrzeilige Eingaben mit textarea erfassen

Dropdowns

                    
                        <select name="kurs">
                            <option>HTML</option>
                            <option>CSS</option>
                            <option>Flexbox</option>
                        </select>
  • Geeignet für eine Auswahl aus einer Liste
  • Spart Platz bei vielen Optionen

Radio-Buttons

                    
                        <input type="radio" name="niveau" value="anfaenger" />
                        <input type="radio" name="niveau" value="mittel" />
                        <input type="radio" name="niveau" value="fortgeschritten" />
  • Für genau eine Auswahl in einer Gruppe
  • Gleicher name verbindet die Gruppe
  • Kein Text/Label per Standart

Checkboxen

                    
                        <input type="checkbox" name="extras" value="handout" />
                        <input type="checkbox" name="extras" value="zertifikat" />
  • Für mehrere unabhängige Auswahlen
  • 0, 1 oder mehrere Optionen können aktiv sein

Textarea

                    
                        <textarea name="bemerkung" rows="5"></textarea>
  • Für längere, mehrzeilige Texte
  • Zum Beispiel Kommentare, Feedback oder Fragen

Wann nehme ich was?

  • Dropdown: eine Auswahl aus vielen Optionen
  • Radio: genau eine sichtbare Auswahl aus wenigen Optionen
  • Checkbox: mehrere Optionen gleichzeitig
  • Textarea: längerer Freitext

Komplettes Beispiel

                    
                        <form action="/feedback" method="post">
                            <select name="thema">
                                <option>HTML</option>
                                <option>CSS</option>
                            </select>

                            <input type="radio" name="delivery" value="Post" />
                            <input type="radio" name="delivery" value="Abholung" />

                            <input type="checkbox" name="extras" value="handout" />

                            <textarea name="kommentar" rows="4"></textarea>

                            <button type="submit">Senden</button>
                        </form>

Label Elemenmt

  • Beschriften Eingabefelder
  • Das Attribute 'for' verknüpft Labels mit der 'id' der Eingabefelder
                    
                        <label for="post">Post</label>
                        <input type="radio" id="post" value="Post" />

                        <label for="pickup">Post</label>
                        <input type="radio" id="pickup" value="Abholung" />

Typische Fehler

  • Radio-Buttons mit verschiedenen name-Werten versehen
  • Checkboxen und Radio-Buttons verwechseln
  • textarea wie ein normales input schreiben
