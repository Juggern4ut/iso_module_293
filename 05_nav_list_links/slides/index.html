Listen, Navigation und Links

Strukturierte Inhalte im Web

Lernziele

  • Listen mit HTML korrekt erstellen
  • Unterschied zwischen ul und ol verstehen
  • Navigationen semantisch richtig umsetzen
  • Saubere und suchmaschinenfreundliche Links erstellen

Warum Listen?

  • Strukturieren Inhalte
  • Verbessern die Lesbarkeit
  • Werden oft für Navigationen verwendet
  • Sind leicht mit CSS zu stylen

Listenelement (li)

Jedes Element innerhalb einer Liste wird mit dem Tag li definiert.

li darf nur innerhalb von ul oder ol verwendet werden.

Navigation im Web

  • Hilft Benutzern, sich zurechtzufinden
  • Ist auf fast jeder Webseite vorhanden
  • Sollte klar und einfach aufgebaut sein

Das nav-Element

HTML bietet mit nav ein semantisches Element für Navigationen.

  • Verbessert Barrierefreiheit
  • Hilft Suchmaschinen
  • Macht den Code lesbarer

Links (a-Element)

Links verbinden Webseiten miteinander.

  • Interne Links
  • Externe Links

Interne und externe Links

  • Interne Links: eigene Dateien
  • Externe Links: andere Webseiten

Best Practices für Linktexte

  • Linktexte sollen beschreibend sein
  • Keine generischen Begriffe verwenden
  • Linktext muss Sinn ergeben

Links und SEO

  • Suchmaschinen lesen Linktexte
  • Gute Struktur verbessert Auffindbarkeit
  • Navigation sollte logisch aufgebaut sein
