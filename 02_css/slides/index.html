Bringt Stil in die Webseite

Lernziele dieser Lektion

  • Verstehen, was CSS ist und warum es verwendet wird.
  • Erkennen, wie Selektoren funktionieren und aufgebaut sind.
  • Die wichtigsten CSS-Eigenschaften kennen und anwenden können.

Was ist CSS?

(Cascading Style Sheets)
  • CSS bestimmt das Aussehen von HTML-Elementen.
  • Farben, Grössen, Abstände, Position und vieles mehr.
  • Kann inline, im <head> oder als externes Stylesheet eingebunden werden.

Gleicher HTML-Code – unterschiedliche CSS-Regeln

Aufbau einer CSS-Regel

CSS Regeln bestehen aus Selektor, Property und Wert:

            
            selector {
              property: wert;
            }
Selektor Property Wert

Selektoren

  • element – Selektiert alle Elemente dieses Typs
  • .klasse – Selektiert alle Elemente mit dieser Klasse
  • #id – Selektiert ein eindeutiges Element
  • element element – Selektiert verschachtelte Elemente
            
            p { color: red; }
            .button { padding: 10px; }
            #header { background: black; }
            nav a { text-decoration: none; }

Wie bindet man CSS ein?

  • Inline: direkt im HTML-Tag
  • Intern: im <style> Block im Head
  • Extern: ausgelagert in .css Datei (Standard)

Das Box-Modell

Jedes HTML-Element wird als Box dargestellt:

  • Content
  • Padding
  • Border
  • Margin

Kaskade & Spezifität

  • CSS entscheidet nach Priorität, welche Regel gilt.
  • ID > Klasse > Element
  • Später definierte Regeln überschreiben frühere.
